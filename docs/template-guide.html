How To Write a portafile.yaml

A portafile.yaml is a simple yaml file used to define the steps and configurations needed to automatically compose a project.

Supported source_template values

  • alpine-3.17: lightweight and fast.
  • centos-9-stream: CentOS/RHEL ecosystem compatibility.
  • opensuse-15.4: openSUSE-based environment.
  • ubuntu-20.04: broad package/tool compatibility.

Available arguments

  • source_template (required)
  • name (optional, pre-fills device name)
  • hostname (optional, pre-fills hostname)
  • instructions / steps (optional command list)
  • expose_ports (optional list of ports)
  • metadata (optional extra values)

Example

source_template: ubuntu-20.04
name: Demo Service
hostname: demo-service
instructions:
  - command: sudo apt-get update
  - command: sudo apt-get install -y docker.io
expose_ports:
  - 8080
